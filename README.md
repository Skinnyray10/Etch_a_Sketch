# Etch a Sketch project 

A browser-based drawing app built with HTML, CSS, and JavaScript.  
Users can create a customizable grid and draw by hovering over squares.

## Features
- Generate dynamic grid (up to 100x100)
- Hover to paint squares
- Color picker support
- Reset grid with custom size

## How to use
1. Open index.html in your browser
2. Select grid size
3. Choose a color
4. Hover over the grid to draw

## Built with
- HTML
- CSS (Flexbox)
- JavaScript (DOM manipulation)

- Event handling
